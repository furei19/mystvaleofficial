            * *      Download the archive       * *

                     Extract the archive:

            For .tar.gz: "tar -xzvf mystvale.tar.gz"
                 For .zip: "unzip mystvale.zip"

         Allow the program to run (if blocked by macOS).

     Go to the extracted folder: "cd /path/to/mystvale-bundle"

                  Run the program: "./mystvale"
